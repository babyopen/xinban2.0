# 域名解析与服务器配置指南

## 一、域名解析配置

### 1. 登录域名管理平台

**常见域名管理平台**：
- 阿里云：https://dc.aliyun.com/
- 腾讯云：https://console.cloud.tencent.com/domain
- 百度云：https://console.bce.baidu.com/domain/
- 其他域名注册商

### 2. 添加域名解析记录

**步骤**：
1. 登录域名管理平台
2. 找到 `xboyi.cn` 域名
3. 进入 `DNS 解析` 管理
4. 添加以下解析记录：

| 记录类型 | 主机记录 | 记录值 | TTL | 说明 |
|---------|---------|--------|-----|------|
| A | @ | 服务器公网IP | 10分钟 | 主域名解析 |
| A | www | 服务器公网IP | 10分钟 | www子域名解析 |

**示例**：
- 主域名：`xboyi.cn` → 解析到 `192.168.1.100`
- www子域名：`www.xboyi.cn` → 解析到 `192.168.1.100`

### 3. 验证域名解析

**方法**：
1. 等待解析生效（通常10-30分钟）
2. 使用 `ping` 命令验证：
   ```bash
   ping www.xboyi.cn
   ping xboyi.cn
   ```
3. 使用 `nslookup` 命令验证：
   ```bash
   nslookup www.xboyi.cn
   nslookup xboyi.cn
   ```

## 二、服务器配置

### 1. 服务器初始化

**登录服务器**：
```bash
# 使用SSH登录
ssh root@服务器公网IP
```

**更新系统**：
```bash
# CentOS 系统
yum update -y

# Ubuntu/Debian 系统
apt update && apt upgrade -y
```

### 2. 安装必要软件

**安装基础工具**：
```bash
# CentOS 系统
yum install -y wget curl git vim net-tools

# Ubuntu/Debian 系统
apt install -y wget curl git vim net-tools
```

**安装 Python 环境**：
```bash
# 安装 Python 3.8+
# CentOS 系统
yum install -y python3 python3-pip python3-devel

# Ubuntu/Debian 系统
apt install -y python3 python3-pip python3-dev
```

### 3. 防火墙配置

**检查防火墙状态**：
```bash
# CentOS 系统
systemctl status firewalld

# Ubuntu/Debian 系统
ufw status
```

**开放端口**：
```bash
# CentOS 系统
firewall-cmd --zone=public --add-port=80/tcp --permanent
firewall-cmd --zone=public --add-port=443/tcp --permanent
firewall-cmd --zone=public --add-port=8000/tcp --permanent
firewall-cmd --reload

# Ubuntu/Debian 系统
ufw allow 80/tcp
ufw allow 443/tcp
ufw allow 8000/tcp
ufw reload
```

### 4. 宝塔面板安装

**执行安装命令**：
```bash
# CentOS 系统
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

# Ubuntu/Debian 系统
wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh
```

**记录登录信息**：
安装完成后，会显示宝塔面板的登录地址、用户名和密码，请妥善保存。

### 5. 宝塔面板初始化

**登录宝塔面板**：
1. 打开浏览器，访问宝塔面板登录地址
2. 输入用户名和密码登录
3. 完成初始化设置

**安装必要软件**：
在宝塔面板的 `软件商店` 中安装：
- **Nginx**：Web 服务器
- **Python项目管理器**：管理 Python 环境
- **MySQL**（可选）：如果需要数据库

## 三、服务器安全配置

### 1. 修改 SSH 端口

**编辑 SSH 配置**：
```bash
vim /etc/ssh/sshd_config
```

**修改端口**：
```
# 将默认端口 22 修改为其他端口，如 2222
Port 2222
```

**重启 SSH 服务**：
```bash
# CentOS 系统
systemctl restart sshd

# Ubuntu/Debian 系统
systemctl restart ssh
```

### 2. 创建普通用户

**创建用户**：
```bash
useradd -m xboyi
passwd xboyi
```

**添加 sudo 权限**：
```bash
usermod -aG wheel xboyi  # CentOS
usermod -aG sudo xboyi   # Ubuntu/Debian
```

### 3. 禁用 root 远程登录

**编辑 SSH 配置**：
```bash
vim /etc/ssh/sshd_config
```

**修改配置**：
```
PermitRootLogin no
```

**重启 SSH 服务**：
```bash
# CentOS 系统
systemctl restart sshd

# Ubuntu/Debian 系统
systemctl restart ssh
```

### 4. 配置 Fail2ban

**安装 Fail2ban**：
```bash
# CentOS 系统
yum install -y epel-release
yum install -y fail2ban

# Ubuntu/Debian 系统
apt install -y fail2ban
```

**配置 Fail2ban**：
```bash
vim /etc/fail2ban/jail.local
```

**添加配置**：
```
[sshd]
enabled = true
port = ssh
filter = sshd
logpath = /var/log/auth.log
maxretry = 5
bantime = 3600
```

**启动 Fail2ban**：
```bash
# CentOS 系统
systemctl start fail2ban
systemctl enable fail2ban

# Ubuntu/Debian 系统
systemctl start fail2ban
systemctl enable fail2ban
```

## 五、常见问题

### 1. 域名解析不生效
- **原因**：DNS 缓存、解析记录未生效
- **解决**：等待 30 分钟后再测试，或使用 `nslookup` 检查解析状态

### 2. 端口无法访问
- **原因**：防火墙未开放、服务未启动
- **解决**：检查防火墙配置，确保端口已开放

### 3. SSH 连接失败
- **原因**：端口修改后未开放、权限问题
- **解决**：检查防火墙配置，确保新端口已开放

### 4. 宝塔面板无法访问
- **原因**：端口未开放、服务器安全组限制
- **解决**：检查服务器安全组配置，确保宝塔面板端口已开放
