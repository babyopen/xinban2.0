# 小摇筛选项目 - 宝塔面板部署指南

## 一、环境准备

### 1. 服务器要求

- **操作系统**：CentOS 7/8 或 Ubuntu 18.04+ 
- **内存**：至少 2GB
- **CPU**：至少 1核
- **磁盘**：至少 20GB 空闲空间
- **网络**：公网IP，开放 80/443 端口

### 2. 宝塔面板安装

如果尚未安装宝塔面板，请执行以下命令：

```bash
# CentOS 系统
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

# Ubuntu/Debian 系统
wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh
```

安装完成后，记录面板登录地址、用户名和密码。

### 3. 域名解析

在域名管理平台（如阿里云、腾讯云等）将域名 `www.xboyi.cn` 解析到服务器的公网IP。

## 二、项目部署

### 1. 上传项目文件

**方法一：使用宝塔面板文件管理**

1. 登录宝塔面板
2. 进入 `文件` 管理
3. 进入 `/www/wwwroot/` 目录
4. 创建 `xboyi.cn` 目录
5. 上传项目所有文件到该目录

**方法二：使用 Git 克隆**

1. 登录服务器
2. 进入 `/www/wwwroot/` 目录
3. 执行：`git clone 项目仓库地址 xboyi.cn`

### 2. 安装必要软件

在宝塔面板的 `软件商店` 中安装：

- **Nginx**：Web 服务器
- **Python项目管理器**：管理 Python 环境
- **MySQL**（可选）：如果需要数据库

### 3. 配置 Python 环境

**方法一：使用 Python项目管理器**

1. 打开 `Python项目管理器`
2. 点击 `添加项目`
3. 配置：
  - 项目路径：`/www/wwwroot/xboyi.cn`
  - Python版本：选择 3.8 或更高
  - 框架：Flask
  - 启动方式：Gunicorn
  - 端口：8000
4. 点击 `确定` 创建项目
5. 进入项目 `设置` → `模块`，安装依赖：
  - 点击 `安装模块` → `从文件导入`
  - 选择 `requirements.txt` 文件
  - 点击 `确定` 等待安装完成

**方法二：手动安装**

1. 登录服务器
2. 进入 `/www/wwwroot/xboyi.cn` 目录
3. 执行：
  ```bash
   # 安装依赖
   pip3 install -r requirements.txt
  ```

### 4. 配置 Nginx

**方法一：使用宝塔面板**

1. 打开 `网站` → `添加站点`
2. 配置：
  - 域名：`www.xboyi.cn`
  - 根目录：`/www/wwwroot/xboyi.cn/frontend`
  - PHP版本：纯静态
3. 点击 `提交`
4. 进入站点 `设置` → `反向代理`
5. 点击 `添加反向代理`
6. 配置：
  - 代理名称：`flask_app`
  - 目标URL：`http://127.0.0.1:8000`
  - 发送域名：`$host`
  - 启用 WebSocket：关闭
7. 点击 `提交`
8. 进入 `配置文件`，替换为 `nginx_config.conf` 中的配置

**方法二：直接编辑配置文件**

1. 登录服务器
2. 编辑 Nginx 配置文件：`/www/server/nginx/conf/vhost/xboyi.cn.conf`
3. 复制 `nginx_config.conf` 中的配置内容
4. 保存并重启 Nginx

### 5. 配置文件权限

1. 登录服务器
2. 进入 `/www/wwwroot/xboyi.cn` 目录
3. 执行：
  ```bash
   # 设置文件权限
   chmod -R 755 .
   chmod +x baota_start.sh

   # 设置目录权限
   chmod 775 logs/
   chmod 775 temp/
  ```

### 6. 启动应用

**方法一：使用启动脚本**

1. 登录服务器
2. 进入 `/www/wwwroot/xboyi.cn` 目录
3. 执行：
  ```bash
   ./baota_start.sh
  ```

**方法二：使用 Python项目管理器**

1. 打开 `Python项目管理器`
2. 找到 `xboyi.cn` 项目
3. 点击 `启动` 按钮

### 7. 配置 SSL 证书（可选）

1. 打开 `网站` → `xboyi.cn` → `SSL`
2. 选择 `Let's Encrypt`
3. 勾选域名 `www.xboyi.cn` 和 `xboyi.cn`
4. 点击 `申请`
5. 申请成功后，修改 Nginx 配置文件，取消注释 HTTPS 配置部分

## 三、功能测试

### 1. 基本访问测试

- 访问：`http://www.xboyi.cn`
- 检查：页面是否正常加载，样式是否正确

### 2. API 测试

- 健康检查：`http://www.xboyi.cn/api/health`
  - 预期：返回 `{"status": "ok", "message": "API is running"}`
- 生肖映射：`http://www.xboyi.cn/api/zodiac-mapping`
  - 预期：返回完整的生肖映射表
- 预测接口：使用 POST 请求测试
  ```bash
  curl -X POST http://www.xboyi.cn/api/predict \
    -H "Content-Type: application/json" \
    -d '{"history": [{"period": "2024001", "zodiac": "马"}, {"period": "2024002", "zodiac": "蛇"}]}'
  ```
  - 预期：返回预测结果

### 3. 功能测试

- 测试前端页面的所有功能
- 测试预测功能是否正常
- 测试数据展示是否正确

## 四、监控与维护

### 1. 日志管理

- **Nginx 日志**：`/www/wwwlogs/xboyi.cn.access.log` 和 `/www/wwwlogs/xboyi.cn.error.log`
- **应用日志**：`/www/wwwroot/xboyi.cn/logs/gunicorn_access.log` 和 `/www/wwwroot/xboyi.cn/logs/gunicorn_error.log`

### 2. 定期维护

- **备份**：定期备份项目文件和数据
- **更新**：定期更新依赖包
- **监控**：监控服务器资源使用情况

### 3. 故障排查

- **502 错误**：检查应用是否运行，端口是否占用
- **404 错误**：检查 Nginx 配置和文件路径
- **500 错误**：检查应用日志，查看具体错误信息

## 五、常见问题

### 1. 依赖安装失败

- 检查 Python 版本是否正确
- 检查网络连接是否正常
- 尝试使用国内镜像源：
  ```bash
  pip3 install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
  ```

### 2. 应用启动失败

- 检查端口 8000 是否被占用
- 检查模型文件是否存在
- 查看应用日志：`logs/gunicorn_error.log`

### 3. 页面显示异常

- 检查 Nginx 配置是否正确
- 检查静态文件路径是否正确
- 检查样式文件是否能正常加载

### 4. API 接口调用失败

- 检查应用是否运行
- 检查 Nginx 反向代理配置
- 检查 API 路径是否正确
