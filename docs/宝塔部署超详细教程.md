## 1️⃣ 准备工作

### 1.1 您需要有这些

✅ 一台服务器（已经装好了宝塔面板）
✅ 域名：www.xboyi.cn（已经解析到服务器）
✅ 我们的项目文件

### 1.2 确认您的项目文件在本地电脑上

在您的本地电脑上，确保有以下文件：

```
新版2.0/
├── index.html          ✅ 这个要上传
├── style.css          ✅ 这个要上传
├── api/               ✅ 这个文件夹要上传
│   └── index.py
├── python/            ✅ 这个文件夹要上传
│   ├── ml_api_server.py
│   └── zodiac_ml_predictor.py
├── zodiac_model.pkl   ✅ 这个要上传
├── lottery_history.csv ✅ 这个要上传
└── docs/              ✅ 这个文件夹要上传（可选）
```

⚠️ **重要提醒**：
- 使用根目录的 `index.html`，**不要**用 `frontend/` 目录下的！
- 我已经帮您修改好了API地址，直接上传就行！

## 2️⃣ 第一步：上传文件到服务器

### 2.1 登录宝塔面板

在浏览器输入您的宝塔面板地址，登录进去。

### 2.2 打开文件管理器

1. 点击左侧菜单的 **「文件」**
2. 进入 `/www/wwwroot/` 目录（如果不在的话）

### 2.3 创建项目文件夹

1. 在 `/www/wwwroot/` 目录下
2. 点击上方的 **「新建」** → **「目录」**
3. 目录名输入：`新版2.0`
4. 点击 **「确定」**
5. 双击进入 `新版2.0` 文件夹

### 2.4 上传文件

现在您在 `/www/wwwroot/新版2.0/` 目录下，开始上传文件：

#### 方法A：压缩上传（推荐，更快）

1. 在本地电脑上，把整个 `新版2.0` 文件夹压缩成 zip
2. 在宝塔文件管理器中，点击 **「上传」**
3. 选择刚才压缩的 zip 文件
4. 等待上传完成
5. 上传完成后，右键点击 zip 文件，选择 **「解压」**
6. 确认解压到当前目录
7. 删除 zip 文件（可选）

#### 方法B：逐个上传

1. 点击 **「上传」**
2. 从本地电脑选择 `index.html`，上传
3. 选择 `style.css`，上传
4. 选择 `zodiac_model.pkl`，上传
5. 选择 `lottery_history.csv`，上传
6. 创建 `api` 文件夹，进入后上传 `api/index.py`
7. 创建 `python` 文件夹，进入后上传 `python/ml_api_server.py` 和 `python/zodiac_ml_predictor.py`

### 2.5 确认文件都上传好了

上传完成后，检查 `/www/wwwroot/新版2.0/` 目录下应该有：

```
✅ index.html
✅ style.css
✅ api/
   └── index.py
✅ python/
   ├── ml_api_server.py
   └── zodiac_ml_predictor.py
✅ zodiac_model.pkl
✅ lottery_history.csv
```

## 3️⃣ 第二步：在宝塔中创建网站

### 3.1 添加站点

1. 点击左侧菜单的 **「网站」**
2. 点击 **「添加站点」** 按钮
3. 填写以下信息：

| 项目 | 填写内容 |
|------|---------|
| 域名 | `www.xboyi.cn`（如果有多个域名，空格分开） |
| 根目录 | `/www/wwwroot/新版2.0` |
| FTP | 不创建 |
| 数据库 | 不创建 |
| PHP版本 | 纯静态 |

4. 点击 **「提交」** 按钮

### 3.2 确认网站创建成功

您应该能在网站列表中看到 `www.xboyi.cn` 了。

## 4️⃣ 第三步：配置Python环境

### 4.1 安装Python项目管理器

1. 点击左侧菜单的 **「软件商店」**
2. 在搜索框输入：`Python`
3. 找到 **「Python项目管理器」**
4. 点击 **「安装」**（如果已经安装了就不用了）

### 4.2 安装Python 3.9

1. 在软件商店，点击 **「Python项目管理器」** 右侧的 **「设置」**
2. 点击 **「版本管理」**
3. 点击 **「安装版本」**
4. 选择 Python 3.9（或者更高版本，比如 3.10）
5. 点击 **「安装」**
6. 等待安装完成（可能需要几分钟）

### 4.3 安装Python依赖包

#### 方法A：在宝塔终端安装（推荐）

1. 点击左侧菜单的 **「终端」**
2. 在终端中输入以下命令，一行一行输：

```bash
cd /www/wwwroot/新版2.0
pip install flask numpy scikit-learn
```

3. 按回车执行
4. 等待安装完成（可能需要几分钟）

#### 方法B：在Python项目管理器中安装

稍后在创建项目时再安装也可以。

## 5️⃣ 第四步：启动后端服务

我们需要启动两个服务：
1. API服务（端口8000）
2. ML服务（端口5001）

### 5.1 创建并启动API服务

1. 点击左侧菜单的 **「软件商店」**
2. 找到 **「Python项目管理器」**，点击 **「设置」**
3. 点击 **「添加项目」**
4. 填写以下信息：

| 配置项 | 填写内容 |
|--------|---------|
| 项目名称 | `小摇筛选-API` |
| 路径 | `/www/wwwroot/新版2.0/api` |
| 启动文件 | `index.py` |
| 端口 | `8000` |
| Python版本 | 选择刚才安装的3.9 |

5. 点击 **「确定」**
6. 如果需要安装依赖，点击 **「安装模块依赖」**
7. 点击 **「启动」** 按钮
8. 等待几秒，状态应该变成 **「运行中」** ✅

### 5.2 创建并启动ML服务

1. 同样在Python项目管理器中
2. 点击 **「添加项目」**
3. 填写以下信息：

| 配置项 | 填写内容 |
|--------|---------|
| 项目名称 | `小摇筛选-ML` |
| 路径 | `/www/wwwroot/新版2.0/python` |
| 启动文件 | `ml_api_server.py` |
| 端口 | `5001` |
| Python版本 | 选择刚才安装的3.9 |

4. 点击 **「确定」**
5. 如果需要安装依赖，点击 **「安装模块依赖」**
6. 点击 **「启动」** 按钮
7. 等待几秒，状态应该变成 **「运行中」** ✅

### 5.3 确认两个服务都在运行

在Python项目管理器中，您应该看到：
- ✅ 小摇筛选-API - 运行中
- ✅ 小摇筛选-ML - 运行中

## 6️⃣ 第五步：配置Nginx反向代理

### 6.1 打开网站配置

1. 点击左侧菜单的 **「网站」**
2. 找到 `www.xboyi.cn`，点击右侧的 **「设置」**
3. 点击 **「配置文件」** 选项卡

### 6.2 修改配置文件

您会看到一些默认的配置，**全部删掉**，然后复制下面的完整配置粘贴进去：

```nginx
server {
    listen 80;
    server_name www.xboyi.cn xboyi.cn;
    root /www/wwwroot/新版2.0;
    index index.html;
    
    # 启用gzip压缩，让网站更快
    gzip on;
    gzip_vary on;
    gzip_min_length 1024;
    gzip_comp_level 6;
    gzip_types text/plain text/css text/xml text/javascript application/json application/javascript application/xml+rss;
    
    # 前端页面
    location / {
        try_files $uri $uri/ /index.html;
        
        # 静态资源缓存30天
        location ~* \.(css|js|jpg|jpeg|png|gif|ico|svg)$ {
            expires 30d;
            add_header Cache-Control "public, immutable";
        }
    }
    
    # 后端API代理（端口8000）
    location /api/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
    
    # ML API代理（端口5001）
    location /ml-api/ {
        rewrite ^/ml-api/(.*) /$1 break;
        proxy_pass http://127.0.0.1:5001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
    
    # 禁止别人访问敏感文件
    location ~ /\. {
        deny all;
        access_log off;
        log_not_found off;
    }
    
    location ~* \.(py|pkl|csv|log)$ {
        deny all;
        access_log off;
    }
    
    # 日志文件位置
    access_log /www/wwwlogs/www.xboyi.cn-access.log;
    error_log /www/wwwlogs/www.xboyi.cn-error.log;
}
```

### 6.3 保存配置

1. 确认粘贴的配置正确
2. 点击 **「保存」** 按钮

## 7️⃣ 第六步：配置HTTPS（推荐，让网站更安全）

### 7.1 申请免费SSL证书

1. 在网站设置页面，点击 **「SSL」** 选项卡
2. 选择 **「Let's Encrypt」**（免费的）
3. 勾选您的域名：`www.xboyi.cn` 和 `xboyi.cn`
4. 点击 **「申请」** 按钮
5. 等待申请完成（可能需要几十秒）

### 7.2 强制HTTPS

1. 申请成功后，在SSL设置页面
2. 找到 **「强制HTTPS」** 选项
3. 勾选它
4. 点击 **「保存」**

现在您的网站会自动跳转到 https://www.xboyi.cn 了！

## 8️⃣ 第七步：配置防火墙

### 8.1 开放必要的端口

1. 点击左侧菜单的 **「安全」**
2. 确保以下端口是 **「已放行」** 的：

| 端口 | 用途 | 状态 |
|------|------|------|
| 80 | HTTP网站 | ✅ 必须放行 |
| 443 | HTTPS网站 | ✅ 必须放行 |

⚠️ **重要**：
- ❌ **不要**放行 8000 端口
- ❌ **不要**放行 5001 端口
- 这两个端口通过Nginx代理访问，不需要对外开放，更安全！

## 🚨 常见问题解决

### 问题1：网站打不开，显示404

**可能原因**：
- 根目录设置错了
- 文件没上传到正确位置

**解决方法**：
1. 检查网站根目录是不是 `/www/wwwroot/新版2.0`
2. 检查 `/www/wwwroot/新版2.0/` 下面有没有 `index.html`

---

### 问题2：网站打开了，但ML预测用不了

**可能原因**：
- ML服务没启动
- Nginx配置错了
- index.html中的API地址没改

**解决方法**：
1. 去Python项目管理器看「小摇筛选-ML」是不是运行中
2. 检查网站配置文件中的 `/ml-api/` 配置
3. 确认用的是根目录的 `index.html`，不是frontend/下的

---

### 问题3：API服务或ML服务启动失败

**可能原因**：
- Python依赖没安装
- 端口被占用
- 路径配置错了

**解决方法**：
1. 检查依赖是不是都安装了：`flask`, `numpy`, `scikit-learn`
2. 看Python项目管理器的日志，有什么错误提示
3. 确认启动文件路径正确

---

### 问题4：页面显示了，但有很多红色错误

**解决方法**：
1. 按F12打开开发者工具
2. 点击Console（控制台）
3. 看红色的错误信息是什么
4. 根据错误信息排查

---

### 问题5：想重启服务

**方法A：在Python项目管理器中**
1. 找到服务
2. 点击「停止」
3. 点击「启动」

**方法B：在终端中（如果是用nohup启动的）**
```bash
# 停止API服务
pkill -f "api/index.py"

# 停止ML服务
pkill -f "ml_api_server.py"

# 重新启动
cd /www/wwwroot/新版2.0
nohup python3 api/index.py > api.log 2>&1 &
nohup python3 python/ml_api_server.py > ml.log 2>&1 &
```
