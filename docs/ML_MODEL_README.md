# 生肖预测2.0 - 机器学习模型使用说明

## 概述

这是一个基于XGBoost的生肖预测机器学习模型，使用历史开奖数据进行训练，能够预测下一期每个生肖的出现概率。

## 文件结构

```
新版2.0/
├── zodiac_ml_predictor.py    # 主程序
├── lottery_history.csv       # 示例数据文件
├── zodiac_model.json         # 训练好的模型（运行后生成）
└── ML_MODEL_README.md        # 本说明文档
```

## 数据格式

### 输入数据文件 (lottery_history.csv)

```csv
period,zodiac
1,3
2,5
3,5
4,8
...
```

- **period**: 期号，整数，唯一且按时间递增
- **zodiac**: 开奖生肖，整数 1-12
  - 1=马, 2=蛇, 3=龙, 4=兔, 5=虎, 6=牛
  - 7=鼠, 8=猪, 9=狗, 10=鸡, 11=猴, 12=羊

## 使用方法

### 1. 训练模型

```bash
python zodiac_ml_predictor.py
```

程序会自动：
1. 加载数据文件
2. 构建特征（150+个特征）
3. 划分训练集(80%)和测试集(20%)
4. 训练XGBoost模型
5. 评估模型性能
6. 预测下一期概率
7. 保存模型文件

### 2. 使用训练好的模型进行预测

```python
from zodiac_ml_predictor import load_model, predict_next
import pandas as pd

# 加载模型
model = load_model('zodiac_model.json')

# 加载历史数据
df = pd.read_csv('lottery_history.csv')

# 预测下一期
probabilities = predict_next(model, df.iloc[-1], df)

# 显示结果
for i, prob in enumerate(probabilities):
    print(f"生肖 {i+1}: {prob:.4f}")
```

## 特征工程

### 3.1 基础统计特征（120个）

每个生肖10个特征：
- 当前遗漏
- 遗漏比例
- 近10/20/50期出现次数
- 近10/20期频率
- 热门排名
- 连开次数
- 连断状态

### 3.2 动态特征（9个）

- 上期生肖
- 位置间隔
- 五行关系（相生/相同/相克）
- 波色相同
- 单双相同
- 大小相同
- 区间相同
- 头数相同
- 尾数相同

### 3.3 时序特征（36个）

每个生肖3个特征：
- 间隔均值
- 间隔标准差
- 热度变化

## 模型参数

- **算法**: XGBoost多分类器
- **目标函数**: multi:softprob
- **类别数**: 12
- **最大深度**: 6
- **学习率**: 0.1
- **树的数量**: 200
- **子采样率**: 0.8
- **特征采样率**: 0.8

## 评估指标

1. **准确率**: 预测正确的比例
2. **Top-3准确率**: 真实生肖在前三概率中的比例
3. **对数损失**: 概率分布的准确性

## 注意事项

1. **数据质量**: 确保历史数据完整、准确，按期号排序
2. **数据量**: 建议至少100期历史数据，数据越多模型越稳定
3. **特征工程**: 所有特征基于历史数据计算，无数据泄露
4. **随机性**: 彩票具有随机性，模型仅供参考
5. **更新频率**: 建议定期使用新数据重新训练模型
